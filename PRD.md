# PRD: Topgun Tamagotchi 🥚

> 클래식 다마고치를 웹에서 구현한 가상 펫 키우기 게임

## 1. 개요

### 1.1 제품 비전
90년대 다마고치의 감성을 웹 브라우저에서 재현. 픽셀아트 캐릭터가 LCD 스크린 안에서 살아 움직이며, 사용자가 먹이고, 씻기고, 놀아주며 키우는 가상 펫 게임.

### 1.2 레퍼런스
- Instagram [@flower_gotchi](https://www.instagram.com/flower_gotchi/) — 클래식 다마고치 비주얼 스타일
- 반다이 다마고치 오리지널 (1996) — 게임 메카닉
- 산리오 x 다마고치 콜라보 — 컬러풀한 캐릭터 디자인

### 1.3 기술 스택
| 항목 | 선택 |
|---|---|
| Framework | Next.js (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS |
| Rendering | HTML5 Canvas (픽셀아트) |
| Storage | localStorage (MVP) |
| Hosting | Vercel |
| Repo | github.com/quanttraderkim/topgun-tamagotchi (private) |

## 2. 게임 라이프사이클

### Phase 1: 알 (Egg)
- 첫 방문 시 화면에 알 하나 표시
- 알을 탭/클릭하면 흔들리는 애니메이션
- 5회 탭 → 금이 감 → 부화 애니메이션
- 부화 시 랜덤 캐릭터 1종 등장 (v1.0: 1종, 이후 확장)

### Phase 2: 아기 (Baby) — v1.5
- 부화 직후 아기 상태
- 기본 인터랙션만 가능 (밥, 청소)
- 일정 시간 후 성장

### Phase 3: 성장 (Adult)
- 풀 인터랙션 가능
- 캐릭터가 LCD 스크린 안에서 자유롭게 걸어다님
- 스탯에 따라 표정/행동 변화

### Phase 4: 시니어 — v2.0
- 장기 플레이 시 나이 들기
- 특별한 이벤트/메시지

## 3. 스탯 시스템

| 스탯 | 범위 | 자연 감소 | 회복 방법 |
|---|---|---|---|
| 🍖 배고픔 (Hunger) | 0~100 | -5/분 | 밥주기 (+20) |
| 😊 행복도 (Happiness) | 0~100 | -3/분 | 놀아주기 (+15) |
| 🧹 청결도 (Cleanliness) | 0~100 | 응가 시 -20 | 청소하기 (100 복구) |
| 💤 에너지 (Energy) | 0~100 | -2/분 | 수면 (자동 회복) |

### 스탯 임계값
- **80~100**: 최상 → 캐릭터 해피 상태
- **50~79**: 보통 → 기본 상태
- **20~49**: 위험 → 슬픈 표정, 느린 움직임
- **0~19**: 위급 → 아픈 상태, 경고 알림

### 시간 보정
- 비접속 시간 동안 스탯 감소 (단, 최소 10까지만 — 접속 안 했다고 죽지는 않음)
- 접속 시 "X시간 동안 비웠어요" 메시지

## 4. 인터랙션

### 4.1 밥주기 (Feed)
- 버튼 클릭 → 밥 아이템이 화면에 등장
- 캐릭터가 밥 쪽으로 이동 → 먹는 애니메이션
- 배고픔 +20 (최대 100)
- 쿨다운: 10초

### 4.2 청소하기 (Clean)
- 💩이 있을 때만 활성화
- 버튼 클릭 → 빗자루 이펙트 → 💩 사라짐
- 청결도 100으로 복구

### 4.3 놀아주기 (Play)
- 버튼 클릭 → 캐릭터가 통통 점프하는 애니메이션
- 행복도 +15 (최대 100)
- 쿨다운: 15초

### 4.4 재우기 (Sleep) — v1.5
- 버튼 클릭 → 불 꺼짐 → zzZ 애니메이션
- 에너지 자동 회복 (초당 +2)
- 수면 중 다른 인터랙션 불가

## 5. 캐릭터 애니메이션

### 5.1 기본 스프라이트
16x16 픽셀 그리드, 프로그래밍 방식으로 렌더링 (Canvas).

### 5.2 애니메이션 상태 목록
| 상태 | 프레임 수 | 트리거 |
|---|---|---|
| idle | 2 | 기본 (깜빡깜빡) |
| walk_right | 4 | 자동 이동 |
| walk_left | 4 | 자동 이동 |
| eat | 4 | 밥주기 액션 |
| happy | 4 | 놀아주기 / 스탯 높을 때 |
| sad | 2 | 스탯 낮을 때 |
| sleep | 2 | 수면 모드 |
| sick | 2 | 스탯 위급 |

### 5.3 자유 이동
- 캐릭터가 LCD 스크린 내에서 좌우로 자유롭게 걸어다님
- 랜덤 방향 전환, 멈추기, 걷기 반복
- 벽(스크린 경계)에 닿으면 방향 전환

---

## 6. 응가 시스템 💩

- 랜덤 간격으로 💩 생성 (60~180초)
- 💩은 화면 하단에 쌓임 (최대 3개)
- 💩 개수에 따라 청결도 감소
  - 1개: 청결도 -20
  - 2개: 청결도 -40
  - 3개: 청결도 -60, 캐릭터 불쾌 표정
- 청소 시 모든 💩 한번에 제거

## 7. UI 레이아웃

```
┌─────────────────────────────────┐
│      🥚 Topgun Tamagotchi       │  ← 타이틀
│                                 │
│    ┌───────────────────────┐    │
│    │  ╭───────────────╮    │    │  ← 기기 케이스 (둥근)
│    │  │               │    │    │
│    │  │  LCD Screen   │    │    │  ← Canvas 렌더링 영역
│    │  │  (캐릭터 여기) │    │    │
│    │  │               │    │    │
│    │  ╰───────────────╯    │    │
│    │                       │    │
│    │  ❤️ 80  😊 65  🧹 90  │    │  ← 스탯 바
│    │                       │    │
│    │   (🍖)  (🧹)  (🎮)   │    │  ← 3개 버튼
│    │                       │    │
│    └───────────────────────┘    │
│                                 │
└─────────────────────────────────┘
```

### 기기 디자인
- 둥근 모서리의 파스텔 컬러 케이스
- LCD 스크린은 약간 녹색빛 (클래식 다마고치 느낌) 또는 컬러 LCD
- 하단에 물리 버튼처럼 보이는 3개 원형 버튼
- 모바일에서는 화면 중앙에 꽉 차게

- 아프면: 얼굴에 X 표시
